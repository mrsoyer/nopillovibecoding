# Landing Page Formation / Online Course — Specificites

## Vue d'Ensemble

Une LP de formation (online course, training, bootcamp, certif) **converti en moyenne 13%** vs 11% pour les LP generales (Leadpages). Mais elle a 4 specificites majeures : (1) **outcome-driven copy** plutot que features, (2) **trust eleve via instructeur**, (3) **transparence du curriculum**, (4) **risk reversal fort** (money-back, free preview). Cette doc cible les LP de formation digitale.

## Les 8 Sections Specifiques Formation

| # | Section | Contenu | Obligatoire |
|---|---------|---------|-------------|
| 1 | **Hero** | Outcome + duree + audience | OUI |
| 2 | **Pour qui ?** | Personas cibles avec criteres | OUI |
| 3 | **Ce que tu vas apprendre** | Outcomes specifiques (skills) | OUI |
| 4 | **Curriculum / Modules** | Detail du programme | OUI |
| 5 | **Instructeur** | Bio + credentials + photo | OUI |
| 6 | **Resultats eleves** | Testimonials avec chiffres | OUI |
| 7 | **FAQ** | 6-10 questions specifiques | OUI |
| 8 | **Pricing + Risk Reversal** | Prix + garantie + bonus | OUI |

## Hero Formation — Pattern Eprouve

```
HEADLINE   : Outcome specifique + duree
             "Master Python in 30 Days"
             "From Zero to Launch: Build Your First SaaS in 12 Weeks"

SUBHEAD    : Pour qui + comment + format
             "For data analysts who want to automate reporting,
              with hands-on projects evaluated by mentors."

CTA        : Action specifique
             "Start My Free Lesson"
             "Reserve My Spot - Cohort starts March 15"

TRUST      : Stats / rating
             "4.8/5 from 7,000+ students"
             "As featured in Forbes, TechCrunch"

VISUAL     : Demo video du cours OU screenshot du dashboard d'apprentissage
```

### Variantes de Headline qui Convertissent

| Pattern | Exemple | Quand l'utiliser |
|---------|---------|-------------------|
| Transformation | "From Beginner to Pro in 90 Days" | Skills techniques |
| Outcome chiffre | "Land Your First $5K Client" | Freelance, business |
| Question engageante | "Want to Conquer Public Speaking?" | Soft skills |
| Promesse de mastery | "Master Figma in 3 Weekends" | Tools / software |
| AI-driven differentiation | "AI-Personalized Learning Path" | Modernite tech |
| Career outcome | "Become a Certified [Role]" | Certifications |

## "Ce que tu vas apprendre" — Outcomes Driven

### Mauvais : feature-driven

```
- 10 video lessons
- 5 hours of content
- Downloadable PDFs
- Quizzes after each module
```

### Bon : outcome-driven (eLearning Industry pattern)

```
By the end of this course, you'll be able to:

- Build a complete e-commerce site from scratch using Shopify
- Set up automated email sequences that convert at 25%+
- Launch ad campaigns with $5/day budget on Meta and Google
- Negotiate prices with suppliers in Mandarin (basics)
- Get your first 100 customers within 30 days post-launch
```

**Regle** : commencer chaque bullet par un **verbe d'action** (Build, Master, Create, Launch, Negotiate, Land, Generate).

## Curriculum / Modules — Transparence

### Pattern Recommande

```
Module 1 : Foundations (Week 1-2)
  - Lesson 1.1: Setup your tools [12 min video]
  - Lesson 1.2: First principles [25 min video + quiz]
  - Lesson 1.3: Common pitfalls [18 min video]
  Project: Build your first hello-world app

Module 2 : Advanced patterns (Week 3-4)
  ...
```

### Pourquoi la transparence convertit

> "Course outline shows exactly what students receive" - Code Academy pattern (Unbounce)

Plus de details = plus de confiance. Cacher le curriculum = signal d'arnaque potentiel.

### Variantes

- **Accordeon collapsible** (Reedsy pattern) si curriculum tres long
- **Video preview** par module (Foundr pattern)
- **Telecharger syllabus PDF** (B2B / corporate training)

## Instructeur — Element CLE en Formation

> "Highlight their bio so users know they can be trusted to lead students through the course." (eLearning Industry)

### Bio Anatomie Ideale

```
[PHOTO professionnelle]

[NOM] - [TITRE / EXPERTISE]

[1 LIGNE D'ACCROCHE]
"15 ans a construire des produits SaaS, 3 exits"

[3-5 BULLETS DE CREDENTIALS]
- Ex-Lead Engineer chez Stripe (5 ans)
- Mentore 200+ devs juniors devenus seniors
- Conferences : ReactConf, JSConf, OSCON
- Auteur de "Clean Architecture in Practice"

[SOCIAL LINKS]
LinkedIn / Twitter / GitHub

[QUOTE PERSONNEL]
"J'ai cree ce cours apres avoir vu 1000 devs se planter sur les memes erreurs."
```

### Pattern "Celebrity Instructor" (Foundr style)

Quand l'instructeur est connu : photo grande, titre style magazine, video presentation.

### Pattern "Domain Expert" (Magnum/Soth style)

Quand l'expertise est niche : credentials lourds, mentions media, awards.

## Pricing — Transparence et Flexibilite

### Affichage du Prix

| Pattern | Quand l'utiliser |
|---------|-------------------|
| Prix unique grand visible | Course one-time |
| 3 tiers (Basic / Pro / Enterprise) | Bundles avec coaching |
| Mensuel + annuel (avec discount) | Subscription / membership |
| Prix barre + nouveau prix | Promo / launch |
| "From $X" si custom | B2B corporate training |

### Patterns gagnants (Copyhackers / 10X Emails)

```
Plan Standard - $497
ou 3 paiements de $179

[ Enroll Now ]

✓ All 10 modules
✓ Lifetime access
✓ Mentor feedback on 3 projects
✓ Private community access
✓ Certificate of completion

30-day money-back guarantee
```

### Anti-pattern : "Contact us for pricing"

Sauf B2B custom, c'est un **drop massif**. Visible price = trust.

## Risk Reversal — Critique en Formation

### Patterns Standards

| Pattern | Effet |
|---------|-------|
| **30-day money-back guarantee** | Standard universel |
| **First lesson free** | Reduce purchase risk |
| **14-day free trial** | Subscription courses |
| **No questions asked refund** | Trust booster |
| **Pay only if you finish** | Differentiation forte |

### Placement

- **Pres du CTA principal** (sous le bouton)
- **Repete dans la FAQ**
- **Dans le footer**

```
[ Enroll Now - $497 ]
30-day money-back guarantee. No questions asked.
We're confident you'll love it.
```

## Urgency & Scarcity (specifique formation)

### Patterns Ethiques

| Pattern | Quand utiliser |
|---------|----------------|
| **Cohort start date** | "Next cohort starts March 15" |
| **Limited seats** | Si cohort-based avec mentoring |
| **Bonus expiring** | "Sign up by Friday : free 1-1 session" |
| **Early bird pricing** | "$497 -> $697 after launch week" |
| **Countdown timer** | Lift documente : 4.1% -> 11.3% (LearnWorlds) |

### Anti-pattern

Fausse urgency permanente ("Only 2 spots left" qui ne change jamais). Detecte = perte trust.

## Format Webinaire / Live (variante)

Quand la LP est pour s'inscrire a un webinaire / masterclass live :

| Element | Specification |
|---------|---------------|
| Date / heure | Multi-timezone affiche |
| Duree | "60-min masterclass" |
| Format | "Live + replay 7 jours" |
| Inscription | Email + prenom seulement |
| Reminder system | "We'll send 3 reminders" |
| Replay availability | Mentionner explicitement |

> "Webinars remain one of the highest-converting formats for courses" (Leadpages 2026 trends)

## FAQ Formation — Top 10 Questions

```
1. Pour qui est cette formation ?
2. Quels sont les prerequis ?
3. Combien de temps faut-il pour la completer ?
4. En combien de temps puis-je voir des resultats ?
5. Comment se passe le support / mentoring ?
6. Y a-t-il un certificat / diplome a la fin ?
7. Que se passe-t-il si je n'ai pas le temps de suivre ?
8. La formation est-elle remboursable ?
9. Puis-je acceder au contenu a vie ?
10. Y a-t-il une communaute d'eleves ?
```

Format **accordeon collapsible** pour ne pas alourdir.

## Specifique Formation B2B Corporate

| Element | Specification |
|---------|---------------|
| Pricing | "From $X / employee" + custom quote |
| Bulk discount | Visible si > 10 seats |
| Compliance | Mention RGPD / SOC2 / ISO si dispo |
| LMS integration | Logos LMS supportes |
| Reporting | Mention dashboards / progress tracking |
| Sales contact | Form rapide ou booking calendar |

## Checklist LP Formation

- [ ] Hero avec outcome + duree + audience
- [ ] Section "Pour qui" avec criteres specifiques
- [ ] "Ce que tu vas apprendre" en outcomes-driven (verbes action)
- [ ] Curriculum detaille (modules + lessons + duree)
- [ ] Bio instructeur avec photo + credentials + quote
- [ ] 5+ testimonials avec resultats chiffres
- [ ] Stats brand prominentes (X+ students, Y rating)
- [ ] Pricing transparent + plans flexibles
- [ ] Money-back guarantee (30 jours min)
- [ ] FAQ avec 8-10 questions specifiques
- [ ] Urgency ethique (cohort date, early bird)
- [ ] Free preview ou first lesson free
- [ ] Multi-CTA (3-5x sur la page)
- [ ] Mobile-first design
- [ ] Page speed < 3s
