# Copywriting Framework — Messaging & Persuasion

## Vue d'Ensemble

Le copywriting est le levier #1 de conversion : changer un seul mot dans un headline ou un CTA peut produire des **lifts de 27 a 104%**. Cette doc consolide les frameworks (Problem-Story-Results-Action, You-language, AIDA, PAS) et les patterns de messaging documentes par CopyHackers, Unbounce, Apexure et eLearning Industry.

## Frameworks Principaux

### Framework 1 : Problem -> Story -> Results -> Action (LearnWorlds)

```
1. PROBLEM    -> Tu galeres avec X
2. STORY      -> Voici comment Y a resolu ca avec notre solution
3. RESULTS    -> Voici les chiffres concrets de Y
4. ACTION     -> Demarre maintenant pour avoir les memes resultats
```

Exemple concret (formation) :
```
PROBLEM : Tu n'arrives pas a sortir au-dela de 200 followers.
STORY   : Sarah etait coincee a 150 abonnes pendant 8 mois.
          Apres notre methode, elle a hit 10K en 3 mois.
RESULTS : +9,850 abonnes / +5 figures de revenu mensuel
ACTION  : [Commence ta transformation - 30j garantis]
```

### Framework 2 : PAS (Problem - Agitate - Solution)

Variant plus emotionnel :
```
1. PROBLEM   : Tu galeres avec X
2. AGITATE   : Si tu continues comme ca, voila ce qui t'attend...
3. SOLUTION  : Voici la methode pour resoudre ca
```

Plus aggressif, marche en e-commerce et formation. Risque : perception manipulative si trop pousse.

### Framework 3 : AIDA Classique

```
A - ATTENTION  -> Hook headline
I - INTEREST   -> Sub + benefits
D - DESIRE     -> Social proof + outcomes
A - ACTION     -> CTA
```

### Framework 4 : The 4 Us (Headline)

Un bon headline doit etre :
```
USEFUL    -> Promesse de valeur claire
URGENT    -> Implique action maintenant
UNIQUE    -> Different de la concurrence
ULTRA-SPECIFIC -> Chiffre, deadline, audience precise
```

Exemple : "Master Python in **30 days** with **100+ exercises** corrected by **ex-Google engineers**".

## You-Language vs We-Language

### Regle d'Or

> "You-oriented vs Product-centric : Athabasca prioritizes visitor self-actualization over institutional bragging." (Unbounce)

| Mauvais (We) | Bon (You) |
|--------------|-----------|
| "We are the leader in..." | "You'll get..." |
| "Our platform helps..." | "You'll save 5 hours/week" |
| "We provide 10 tools" | "You'll have everything to launch" |
| "Our students learn..." | "You'll master Python" |

### Calcul rapide

Comptez les "**you / your / yours / yourself**" vs "**we / our / ours / us**" dans votre LP. Ratio cible : **3:1 minimum** en faveur du "you".

## Patterns de Headline qui Convertissent

### Pattern 1 : Outcome + Time

```
"[Outcome specifique] in [duree]"
-> "Master Python in 30 Days"
-> "Build Your First SaaS in 12 Weeks"
-> "Get Paid to Play with Pets" (Rover)
```

### Pattern 2 : You-focused promise

```
"[You verb] [outcome]"
-> "Build something beautiful" (Squarespace)
-> "Start writing today" (Medium)
-> "Take control of your finances"
```

### Pattern 3 : Problem-acknowledgment

```
"[Pain point question or statement]"
-> "Poor credit score?" (HomeLoanGurus)
-> "Tired of 4-hour meetings?"
-> "Want to write but can't find the time?"
```

### Pattern 4 : Specific Number + Audience

```
"[Number] + [outcome] + [audience]"
-> "10,000+ designers use Figma weekly"
```

### Pattern 5 : Audience-first

```
"For [audience] who [pain]"
-> "For freelancers who hate invoicing"
```

## CTA Copywriting

### Regles

```
[Power verb] + [first-person] + [specific benefit]
-> "Get my free SEO audit"
-> "Start my 14-day trial"
-> "Book my 15-min call"
-> "See how it works"
```

### Tableau Power Verbs par Funnel Stage

| Stage | Power Verbs | Exemple |
|-------|-------------|---------|
| **Awareness** | Discover, Learn, Read, See | "See how it works" |
| **Consideration** | Compare, Check, Try | "Try us free" |
| **Decision** | Get, Start, Book, Reserve | "Get my report" |
| **Action** | Buy, Order, Subscribe, Enroll | "Enroll today" |
| **Retention** | Renew, Upgrade, Refer | "Refer a friend" |

### Microcopy Recipes (sous le bouton)

"No credit card required" / "Takes 60 seconds" / "30-day money-back guarantee" / "Cancel anytime" / "Join 10,000+ teams" / "Free for 14 days" / "256-bit SSL encryption"

## Voice & Tone

### Ton selon le contexte

| Contexte | Ton recommande | Exemple |
|----------|----------------|---------|
| **Tech / Dev tools** | Direct, confiant, sans fluff | "Ship code faster" |
| **Formation / Course** | Encouragement, accessible | "Let's get you started" |
| **B2B Enterprise** | Professionnel, ROI-focused | "Reduce costs by 30%" |
| **Lifestyle / Wellness** | Chaleureux, aspirationnel | "Find your calm" |
| **E-commerce premium** | Sophistique, exclusif | "Crafted for connoisseurs" |
| **SaaS startup** | Decontracte, audacieux | "Stop spreadsheet hell" |

### Authenticity Markers

- Eviter le jargon corporate ("synergize", "leverage", "actionable")
- Parler comme votre client parle (focus group / reviews / forums)
- Specifique > generique ("17 minutes" > "quickly")
- Preuves concretes > superlatifs ("3X faster" > "lightning fast")

## Patterns de Persuasion

### Trigger 1 : FOMO (Fear Of Missing Out)

```
"Only 12 spots left in this cohort"
"Bonus expires Friday at midnight"
"Reserve early access" (Vessi pattern)
```

### Trigger 2 : Social Proof Avant Action

```
"Join 50,000+ teams using X"
"As featured in Forbes, TechCrunch"
"4.8/5 from 7,000+ reviews"
```

Place pres du CTA pour reduire l'anxiete.

### Trigger 3 : Risk Reversal

```
"30-day money-back guarantee"
"No credit card required"
"Try free, cancel anytime"
```

Toujours coller au CTA principal.

### Trigger 4 : Specificity / Concretness

```
Vague   : "Save time"
Concret : "Save 5 hours per week"

Vague   : "Many users"
Concret : "5,247 designers"

Vague   : "Quickly"
Concret : "In 17 minutes"
```

### Trigger 5 : Pre-emption d'objection

```
"Free to download" (Honey)         -> objection budget
"No credit card required"          -> objection commitment
"Beginners welcome"                -> objection skill
"Join in 60 seconds"               -> objection time
```

## Structure de Copy par Section

| Section | Longueur cible | Composition |
|---------|---------------|-------------|
| Hero | 50-80 mots | Headline + sub + CTA + microcopy + 1 trust signal |
| Problem | 80-150 mots | Phrase douleur + 2-3 symptomes + question rhetorique |
| Solution | 100-200 mots | Phrase solution + 3-5 benefits + visual + CTA secondaire |
| Social Proof | 100-150 mots | 3-6 testimonials + stats brand |
| Pricing | 50-100 / tier | Nom + prix + alt + 5-7 features + CTA + risk reversal |
| FAQ | 50-80 / question | Question client + reponse directe (pas corporate) |

Exemple Problem Section :
```
Tu passes plus de temps a chercher dans tes notes qu'a creer.

- 47 fichiers Notion eparpilles sans logique
- Cette idee geniale d'il y a 6 mois ? Introuvable.
- Chaque article te demande 3h de recherche dans tes archives.

Et si ton "second cerveau" travaillait pour toi ?
```

## Anti-Patterns Copy

| Anti-pattern | Correction |
|--------------|-----------|
| "We are the leader in X" | "You'll get X" |
| Corporate jargon | Langage du client |
| Headlines vagues / clever | Specifique + benefit-driven |
| Listes de features | Listes de outcomes |
| Superlatifs vides ("amazing") | Chiffres + temoignages |
| CTA "Submit" / "Click here" | "Get my free X" |
| Ratio We:You > 1:1 | Ratio You:We > 3:1 |
| "Quickly" / "easy" | Chiffres specifiques |
| Headlines > 12 mots | Reduire a 6-8 |
| Pas de microcopy sous CTA | Reassurance / objection-removal |

## Process de Copywriting

### Etape 1 : Research client (avant d'ecrire)

- Lire 50+ reviews / temoignages clients
- Noter les **mots exacts** qu'ils utilisent
- Identifier 5 douleurs principales
- Identifier 5 outcomes recherches
- Constituer un "swipe file" de phrases clients

### Etape 2 : Wireframer le copy

- Ecrire H1 + H2 par section AVANT le body
- Verifier que la sequence H1+H2 raconte une histoire complete
- Si on lit que les titres, on doit comprendre l'offre

### Etape 3 : Draft et trim

- Ecrire ronflant puis couper 30%
- Eliminer "very", "really", "just", "actually"
- Remplacer adjectifs par chiffres
- Verifier ratio You:We

### Etape 4 : A/B Test

Tests prioritaires (par impact) :
1. Headline (lift 27-104%)
2. CTA copy (lift jusqu'a 104%)
3. Sub-headline
4. Microcopy CTA
5. FAQ ordering
