# Overview — Principes & Metriques Cles 2026

## Vue d'Ensemble

Une landing page (LP) est une page autonome avec **un seul objectif de conversion**. Sa performance se mesure au taux de conversion (CR) qui depend de 4 leviers : friction (forms, vitesse), clarte (headline, value prop), confiance (social proof) et action (CTA). Les benchmarks 2026 placent le CR moyen autour de 4-11% selon le secteur, avec les LP de formation qui surperforment a 13%.

## Principes Fondamentaux

### 1. Single Goal (1 LP = 1 objectif)
Ajouter un 2eme objectif peut faire **chuter les conversions de 266%** (donnee Unbounce/Zoho). Une LP sert a UNE chose : signup, demo, achat, lead. Tout autre objectif merite une LP separee.

### 2. Clarity > Cleverness
Les headlines avec wordplay perdent face a la clarte. Si le visiteur doit "comprendre" votre offre, il part. Tout doit etre comprehensible **en 3-5 secondes** (delai pour l'utilisateur de former une opinion : 50 millisecondes selon eLearning Industry).

### 3. Above-the-Fold est sacre
Headline + sub + CTA visibles SANS scroll. 53% des utilisateurs mobile abandonnent si load > 3s. La hierarchie obligatoire : **headline first, CTA second, visual third**.

### 4. Friction Removal
> "The brands that win in 2026 are not the ones spending the most, but the ones removing the most friction before they spend." (CRO Digital Marketing)

Friction = champs de form, etapes, ambiguite, vitesse, doute. Chaque friction enlevee = conversion gagnee.

### 5. Mobile-First Reality
83% du trafic vient du mobile, mais le mobile convertit 2x moins que desktop. Donc : **design mobile-first absolu**, mais optimisations desktop pour ne pas perdre les meilleurs convertisseurs.

## Architecture Mentale d'une LP

```
HOOK (hero)         -> j'attire l'attention
PROBLEM (pain)      -> je nomme leur douleur
SOLUTION (offer)    -> je presente ma reponse
PROOF (social)      -> je prouve que ca marche
DETAILS (features)  -> j'explique le contenu
OBJECTIONS (FAQ)    -> je leve les freins
ACTION (CTA final)  -> je convertis
```

Cette sequence repond aux 5 questions silencieuses du visiteur :
1. C'est quoi ? (hero)
2. C'est pour moi ? (problem)
3. Pourquoi vous ? (solution + proof)
4. C'est combien et comment ? (details + pricing)
5. Et si... ? (objections)
