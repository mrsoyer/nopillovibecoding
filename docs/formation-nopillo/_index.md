## TL;DR

**Stack pedagogique** :
- Webflow + Webflow MCP (base)
- Claude Code + skills (`doc-maker`, `cdc-maker`, `skill-maker`)
- MCPs : Webflow, HubSpot, Google Ads, Meta Ads (3 derniers en optionnel selon format)
- Alternative landing : Supabase Edge Functions + Netlify (format 3j uniquement)

**Methodologie centrale** : "Documentation-First + Workflow Industrialise"
1. Enqueter (`/doc-maker`) sur concurrents, design system a aspirer, tech utilisees
2. Cadrer (`/cdc-maker`) le projet en cahier des charges decoupe
3. Capitaliser (`/skill-maker`) les workflows recurrents (form HubSpot, design system Nopillo, landing Google Ads)
4. Executer via MCP (Webflow + autres) avec Claude Code

**Choix de format conseille** :
- **1 jour** : decouverte + 1 livrable concret (recommande pour evaluer)
- **2 jours** : recommande standard, autonomie sur les workflows recurrents
- **3 jours** : pour equipe complete + couvre le "et si on sort de Webflow ?"
