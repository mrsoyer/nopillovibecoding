# 07 — Evaluation : skill check pre + post + NPS

> Outils complets pour evaluer la formation 2 jours selon le modele Kirkpatrick (Niveaux 1 a 4). A envoyer en J-3 (pre), J0 fin (post + NPS), J+30 (Niveau 3), J+90 (Niveau 4).

## Vue d'ensemble

| Phase | Outil | Objectif | Quand |
|-------|-------|----------|-------|
| Pre | Skill check pre | Mesure baseline competence | J-3 |
| Post | Skill check post + NPS | Mesure delta + reaction | J2 17h00 |
| Niveau 3 | Comportement | Application reelle | J+30 |
| Niveau 4 | Resultat | Impact business | J+90 |

Format : Tally / Typeform / Google Forms (5-10 min selon phase).

## 1. Skill check PRE-formation (J-3)

A envoyer 3 jours avant le J1, deadline reponse J-1.

### Section A — Identite

- Prenom + Nom
- Role chez Nopillo
- Anciennete chez Nopillo

### Section B — Auto-evaluation competences (echelle 1-5)

```

## 4. Niveau 4 (J+90)

Mail ou call court, voir `docs/pedagogie-formation/06-evaluation.md`.

```

Q1. Combien de projets / livrables as-tu produits avec la methode apprise ?
[nombre]

Q2. Quel temps estimes-tu avoir gagne (en heures cumulees) ?
[nombre]

Q3. Quel impact business mesurable cite ? (CA, conversion, cout reduit)
[texte libre]

Q4. Sur 1 a 10, le ROI personnel de la formation pour toi ?
[1...10]

Q5. Recommanderais-tu une suite (formation 3j, formation autres equipes Nopillo) ?
[ ] Oui (et pourquoi)
[ ] Non (et pourquoi)

### KPIs Niveau 4

| KPI | Cible Nopillo |
|-----|---------------|
| Temps gagne moyen / participant | > 20h cumulees / mois |
| Nombre projets livres avec methode | > 5 / participant |
| ROI auto-evalue moyen | > 7/10 |

## Anti-patterns evaluation

Reproduit depuis `docs/pedagogie-formation/06-evaluation.md` :

| Anti-pattern | Pourquoi mauvais | Fix |
|--------------|------------------|-----|
| QCM unique fin de session | Mesure memoire pas skill | Skill check + livrable |
| NPS seul | "Sourire" pas "competence" | Niveaux 2-3-4 obligatoires |
| Pas de mesure pre | Pas de delta calcule | Skill check pre + post |
| Pas de J+30 | Niveau 3 ignore | Mail systematique |
| Grille subjective | Variabilite formateur | Criteres objectifs / score |
| Skill check post envoye apres J2 | Taux reponse < 50% | Bloquer 5 min en J2 17h00 |
