## Anti-patterns formateur

| Anti-pattern | Consequence | Fix |
|--------------|-------------|-----|
| Trop de slides theorie | Endormissement, NPS bas | Cap 1h theorie / jour max |
| Perso le clavier sur les exos | Pas d'apprentissage | 1-to-1 = suggerer, pas faire |
| Reagir tard sur ceux qui decrochent | Decrochage definitif | Rondes toutes les 20 min |
| Passer le skill check post a la trappe | Pas de delta = pas de KPI | Bloquer 5 min slide 38 obligatoire |
| Mail post tardif | Perte momentum + NPS chute | Mail J+1 obligatoire avant 12h |
