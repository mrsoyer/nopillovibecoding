## Methodologie de cette doc

1. **7 recherches WebSearch paralleles** sur angles : doc officielle, changelog, config, plugins, themes, fragments/transitions, export PDF
2. **14 extractions WebFetch paralleles** sur les URLs officielles revealjs.com + GitHub releases
3. **Synthese croisee** : informations recoupees entre 2-3 sources minimum
4. **Convergence** sur les faits etablis, divergences notees explicitement

## Limites de cette documentation

- **Pas couvert en profondeur** : plugins tiers, integrations specifiques (Quarto, VuePress, Sphinx)
- **Couvre v5.x uniquement** : pour v6, voir [revealjs.com/upgrading](https://revealjs.com/upgrading/)
- **Pas de benchmarks** : performance variation selon nombre de slides/medias
