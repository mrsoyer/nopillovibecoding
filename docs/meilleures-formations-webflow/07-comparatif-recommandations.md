## Recommandations par profil

### 🎓 Étudiant ou autodidacte curieux
**Budget** : 0-50 €
**Objectif** : découvrir, apprendre les bases

**Recommandation** :
1. **Webflow University** — parcours débutant complet (4-6h)
2. **Webflow 101** + **Welcome to Webflow** + **Make your site responsive**
3. Optionnel : **Udemy** (Hidoun, 20 €) pour un cours en français

**Pourquoi** : aucun investissement, contenu officiel toujours à jour, certif Webflow Practitioner accessible si motivé. Pas besoin de mentorat à ce stade.

---

### 🇫🇷 Salarié français en CDI/CDD voulant se reconvertir
**Budget** : 0 € de poche (CPF + employeur ou France Travail)
**Objectif** : certif RNCP + skills marketables

**Recommandation** :
1. **One Learn** (RNCP37805) — référence absolue avec mentorat illimité 1 an
2. Alternative si profil senior digital : **Maestro** (RNCP41143, no-code + IA, plus large)
3. Si vous voulez un cadre cohorte court : **Le Wagon**

**Pourquoi** :
- Le CPF est la seule certif reconnue par France Travail
- One Learn affiche **92% de réussite** et **85% d'employabilité** post-formation
- Mentorat 1 an = vraie continuité après la formation

**Action** :
1. Vérifier solde CPF : [moncompteformation.gouv.fr](https://www.moncompteformation.gouv.fr)
2. Demander devis One Learn
3. Constituer dossier CPF en ligne

---

### 🥷 Demandeur d'emploi
**Budget** : 0 € (France Travail + CPF)
**Objectif** : skills + certif rapide pour retour à l'emploi

**Recommandation** :
1. **One Learn** (intensif possible avec coaching emploi inclus)
2. **Le Wagon** (cohorte cadrée, alumni network actif)

**Pourquoi** :
- AIF (Aide Individuelle à la Formation) finançable par France Travail
- One Learn inclut 3 sessions de coaching RH + accès job board

**Action** : prendre RDV conseiller France Travail avec devis formation en main.

---

### 💼 Indépendant / TNS qui veut monter en gamme
**Budget** : 1 000-4 000 € (FIFPL/AGEFICE/FAFCEA + CPF)
**Objectif** : ajouter Webflow à son offre client

**Recommandation** :
1. **One Learn** ou **Cube** (mentor dédié)
2. **Flux Academy Masterclass** (Pro Track 1 195 USD) si anglais OK et budget perso

**Pourquoi** :
- OPCO indépendant prend en charge la formation
- Mentor dédié essentiel pour transposer aux projets clients
- Flux Academy = vraie pédagogie projet portfolio

---

### 🏃 Freelance ambitieux carrière internationale
**Budget** : 695 - 3 495 USD
**Objectif** : devenir Webflow Expert reconnu, clients internationaux

**Recommandation séquentielle** :
1. **Webflow University** (gratuit, fondations)
2. **Flux Academy Masterclass** (Pro Track ou Career Ignite Bundle)
3. **Webflow Practitioner Certification** (passage Certiverse)
4. **Build 3 client sites** en parallèle
5. **Webflow Visual Developer Certification**
6. Postuler **Webflow Certified Partner** (commission 20% + listing directory)

**Pourquoi** :
- Flux Academy est **#1 mondial** selon Flowninja
- Le path Certified Partner permet l'accès aux clients premium
- Listing dans le directory officiel Webflow = leads inbound

---

### 🎨 Designer UX/UI qui veut intégrer ses maquettes
**Budget** : 19-695 USD
**Objectif** : autonomie sur l'intégration sans dépendre d'un dev

**Recommandation** :
1. **Design+Code** (19 USD/mois) — Meng To, designer reconnu
2. **Flux Academy Masterclass** si vous voulez aller loin

**Pourquoi** :
- Design+Code est conçu **pour designers** (vous parlez la langue)
- Bundle 50+ cours, valeur exceptionnelle
- Flux ensuite si vous voulez monter en gamme

---

### 📈 Marketer / Growth qui veut produire des landings
**Budget** : 0-100 €
**Objectif** : créer/itérer des landings sans dev

**Recommandation** :
1. **Webflow University** parcours marketer :
   - Webflow 101
   - Webflow SEO fundamentals
   - **AEO from Webflow** (search IA 2026)
   - **Welcome to Optimize** (A/B testing)
   - **Forms**
2. **Webflow MCP + Claude** pour générer du CMS à l'échelle (voir [06-webflow-mcp-claude.md](06-webflow-mcp-claude.md))

**Pourquoi** :
- Vous n'avez pas besoin d'apprendre les composants à fond
- Vous avez besoin de **conversion + SEO + analytics**
- Le MCP Claude vous fait gagner 4-6h par audit SEO (voir use cases)

---

### 👨‍💻 Développeur / Tech profile
**Budget** : Variable
**Objectif** : utiliser Webflow comme outil parmi d'autres

**Recommandation** :
1. **Webflow University** parcours dev : Custom code in Webflow + Intro to integrations
2. **Webflow MCP** dans Claude Code — automatiser opérations
3. **Webflow Visual Developer Certification** si vous travaillez en agence

**Pourquoi** :
- Pas besoin de course longue, vous comprenez vite
- Le MCP est l'outil le plus puissant pour vous (skip API calls manuels)
- La certif Visual Developer est utile en context agency/freelance

---

### 🏢 Agence digitale / équipe
**Budget** : 1 905 - 3 900 € HT/personne (OPCO)
**Objectif** : monter une équipe Webflow rapidement

**Recommandation** :
1. **M2I Formation** ou **Optédif** — formats entreprise, présentiel possible
2. Pour les juniors : **Le Wagon** en cohorte
3. Pour le lead/architect : **Flux Academy Pro Track**
4. Postuler **Webflow Certified Partner** (Enterprise Partner si grosse agence)

**Pourquoi** :
- M2I/Optédif s'adaptent au calendrier équipe
- OPCO de la branche prend en charge
- Status Certified Partner = listing + commission 20%

## Path complet débutant → expert

Voici un plan de progression sur **12 mois** pour passer de zéro à Webflow Certified Partner :

```
MOIS 1-2 : FONDATIONS GRATUITES
├─ Webflow University : Welcome + Webflow 101 + Responsive
├─ Build : 1 site perso simple (portfolio)
└─ Coût : 0 €

MOIS 3-4 : APPROFONDISSEMENT
├─ One Learn (FR/CPF) OU Flux Masterclass (EN)
├─ Build : projet 1 du programme
└─ Coût : 0 € (CPF) ou 695 USD (Flux)

MOIS 5-6 : CERTIFICATION OFFICIELLE
├─ Passer Webflow Practitioner Certification (Certiverse)
├─ Build : 1 site client réel (ami, asso, side-project)
└─ Coût : examen + retake éventuel

MOIS 7-9 : EXPÉRIENCE PRATIQUE
├─ Build : 2 sites clients supplémentaires (3 total)
├─ Suivre Flux Pro Track avancé OU Webflow University Pro
├─ Étudier le Site Grading Rubric Webflow
└─ Coût : variable selon path

MOIS 10-12 : CERTIFIED PARTNER
├─ Passer Webflow Visual Developer Certification
├─ Pre-qualification assessment Certified Partner
├─ Postuler Professional Partner (avec 3 sites portfolio)
├─ Listing directory + commission 20%
└─ Coût : examens (Visual Developer + cert Partner)

RÉSULTAT
├─ Webflow Visual Developer Certified ✓
├─ Webflow Certified Partner ✓
├─ Listed in official directory ✓
├─ Commission 20% sur référents ✓
└─ Salaire freelance : 60-120K USD/an
```

## Décision en 3 questions

**Question 1 : Êtes-vous en France et voulez-vous mobiliser le CPF ?**
- ✅ Oui → **One Learn** (référence) ou **Maestro** (si profil digital senior)
- ❌ Non → question 2

**Question 2 : Avez-vous un budget direct (sans financement) ?**
- ✅ Oui, < 100 € → **Webflow University** + Udemy
- ✅ Oui, 500-1 500 USD → **Flux Academy Masterclass**
- ✅ Oui, 3 000+ USD → **Flux Career Ignite Bundle**

**Question 3 : Quel est votre objectif final ?**
- 🎯 Découvrir/hobbie → **Webflow University**
- 🎯 Reconversion → **One Learn** (FR) ou **Flux Masterclass** (EN)
- 🎯 Freelance carrière → **Flux Pro Track** + **Webflow Certified Partner**
- 🎯 Agence/équipe → **M2I/Optédif** + Certified Partner Enterprise
- 🎯 Marketer landing → **Webflow University parcours marketer** + **MCP Claude**
