## Pourquoi commencer ici

**Webflow University** ([university.webflow.com](https://university.webflow.com/)) est :

- **Gratuit** — aucun paiement requis
- **Officiel** — créé et maintenu par Webflow
- **Toujours à jour** — reflète les dernières features (Optimize, AEO, Analyze)
- **Structuré** — du débutant à l'avancé
- **Évalué** — quizzes intégrés pour mesurer la compréhension

Verdict du comparatif Flowninja (#2 mondial) : « always-up-to-date lessons directly from the platform creators, but lacks structured progression and professional feedback necessary for career advancement ».

**Conclusion** : c'est une **base obligatoire**, mais insuffisante seule pour faire carrière.

## Parcours recommandés

### Parcours 1 — Débutant complet (4 à 6h)
Pour quelqu'un qui n'a jamais touché Webflow :

1. **Welcome to Webflow** (14 min) — comprendre la plateforme
2. **Getting started with Webflow** (1h17) — premiers pas
3. **Webflow 101** (2h28) — fondamentaux + 1 vraie landing page construite
4. **Make your site responsive** (22 min)

À la fin, vous savez construire une landing page simple et la publier.

### Parcours 2 — Designer / Intégrateur (8 à 10h)
Pour quelqu'un qui veut maîtriser le styling et les animations :

1. Parcours débutant complet
2. **CSS layout & positioning** (1h34)
3. **Design systems in Webflow** (19 min)
4. **Interactions & animations** (51 min)
5. **Custom code in Webflow** (57 min)

À la fin, vous pouvez livrer un site visuellement polished.

### Parcours 3 — Marketer / SEO (4 à 5h)
Pour quelqu'un qui veut driver de la conversion :

1. **Welcome to Webflow** + **Webflow 101**
2. **Webflow SEO fundamentals** (31 min)
3. **AEO from Webflow** (1h10) — IA et search 2026
4. **Welcome to Optimize** (25 min) — A/B testing
5. **Intro to Webflow Analyze** (15 min) — mesure
6. **Forms** (1h11) — captation lead

À la fin, vous gérez le funnel de conversion d'une landing.

### Parcours 4 — CMS / Content (3h)
Pour gérer du contenu dynamique :

1. **Design & manage CMS content** (29 min)
2. **Edit content in Webflow** (27 min)
3. **Localize your site** (39 min)
4. **Collaborate with your team** (1h18)

### Parcours 5 — Freelance Bootcamp
Webflow University propose **The Freelancer's Journey** : un cours bootcamp complet pour freelancer en webdesign — pricing, propositions, gestion client, content strategy.

## Certifications officielles Webflow

**Page** : [university.webflow.com/certifications](https://university.webflow.com/certifications)

### Deux certifications disponibles

| Certification | Public | Format | Validité |
|---------------|--------|--------|----------|
| **Webflow Practitioner** | Débutant — valide la compréhension des features clés | 50 QCM | 2 ans |
| **Webflow Visual Developer** | Avancé — valide la capacité à construire des sites scalables prod-ready | 50 QCM | 2 ans |

### Format d'examen
- **50 questions** à choix multiple
- **Surveillance IA** via Certiverse
- **Online** depuis chez vous
- Requiert : ordinateur, caméra, navigateur compatible
- **Retakes** : illimités, mais 24h+ entre tentatives, frais d'inscription complet à chaque fois
- Badge digital partagé via Accredible

### Bénéfices
- Crédibilité industrielle reconnue
- Différenciation sur le marché
- Confiance client
- Préalable au programme **Certified Partner**
- **Webflow University Pro** inclut 10 vouchers d'examen gratuits par an

## Webflow Certified Partner — Le Niveau Pro

**Page** : [webflow.com/certified-partners/apply](https://webflow.com/certified-partners/apply)

C'est le **graal** pour un freelance ou agence Webflow.

### Deux niveaux

| Niveau | Cible |
|--------|-------|
| **Professional Partner** | Freelances, agences digitales, consultants, petites/moyennes entreprises |
| **Enterprise Partner** | Agences travaillant avec grandes entreprises |

### Prérequis pour postuler
- Plan **Freelancer ou Agency Workspace** actif
- **Pre-qualification assessment** réussi (démontre l'expertise Webflow)
- **3 sites client** construits from scratch dans Webflow comme membre Foundations
- Portfolio démontrant un éventail de skills + respect du **site grading rubric**

### Process certification
- Examens administrés par Certiverse, AI proctored
- Certificat valide **2 ans**
- Renouvellement obligatoire avant expiration

### Bénéfices Certified Partner
- **Commission jusqu'à 20%** sur les abonnements Webflow référencés
- **Listing dans le directory officiel** consulté par les entreprises mondiales
- **Client-matching service** : Webflow vous envoie des leads
- **Early access** aux beta features
- Invitations événements exclusifs

### Path complet recommandé

```
Webflow Univ. → Webflow Practitioner cert → Build 3 client sites
        ↓
Webflow Visual Developer cert → Webflow Certified Partner
        ↓
Listed in directory → Client matching → 20% commission
```

## Limites de Webflow University

**À reconnaître honnêtement** :

| Limite | Impact |
|--------|--------|
| Pas de mentorat | Vous êtes seul face aux blocages |
| Pas de feedback sur vos projets | Difficile de savoir si vous progressez bien |
| Progression non structurée pour carrière | Pas de path freelance/agence |
| Pas reconnue France Travail / pas CPF | Aucune valeur officielle en France |
| Pas de communauté intégrée | Vous devez chercher ailleurs (Discord, forum) |

**Verdict pratique** : utiliser Webflow University comme **fondation gratuite**, puis investir dans :
- Une **formation FR certifiante** si vous voulez le CPF + carrière France
- **Flux Academy** si vous voulez la meilleure pédagogie projet
- **Webflow Certified Partner** quand vous avez 3 sites portfolio
