# Vue d'Ensemble — Formation Webflow 2026

## Pourquoi se former à Webflow

Webflow est une plateforme **no-code visuelle** qui permet de concevoir et publier des sites web professionnels sans coder, tout en générant du HTML, CSS et JavaScript propres et performants. C'est un outil de référence pour :

- Les **landing pages haute conversion** (jusqu'à 7 breakpoints custom, performance native)
- Les **sites SaaS et e-commerce**
- Le **freelancing web design** (forte demande, +40% de croissance constatée en 2023)
- Les **agences digitales** qui veulent livrer vite sans dette technique

Salaires moyens d'un développeur Webflow : **60 000 à 120 000 USD/an** aux US ; les freelances expérimentés peuvent dépasser ce plafond.

## Trois grands axes couverts par cette doc

### 1. Les formations
Trois familles distinctes :

| Famille | Exemples | Avantages | Inconvénients |
|---------|----------|-----------|---------------|
| **Officielle gratuite** | Webflow University | Toujours à jour, gratuit, certif officielle | Pas de mentorat, pas structuré pour carrière |
| **Française certifiante** | One Learn, Maestro, Le Wagon, Cube | CPF/OPCO/France Travail, certif RNCP | Plus chère hors financement |
| **Internationale premium** | Flux Academy, Design+Code | Mentorat 1:1, communauté, projets réels | Coût direct, anglais |

### 2. Les landing pages
Une **landing page** est une page d'atterrissage avec un objectif unique de conversion. En 2026, les leviers les plus puissants sont :

- **Réduction des champs de formulaire** : +120% de conversion
- **Optimisation du headline** : +27 à +104% de conversion
- **Mobile first** : 83% du trafic mais conversion 40-51% inférieure au desktop
- **Page speed** : 0.1s gagnée = +8 à 10% de conversion

Détails complets dans [05-landing-pages-best-practices.md](05-landing-pages-best-practices.md).

### 3. Le MCP Webflow + Claude
Lancé le **9 février 2026**, le connecteur Webflow MCP permet à Claude de manipuler directement un site Webflow via langage naturel :

- **Designer API** : modifier la structure, le styling, les éléments
- **Data API** : opérations CMS bulk, audits SEO, gestion de collections
- Compatible avec **Claude Desktop, Claude Code, Cursor, Windsurf**

Détails techniques dans [06-webflow-mcp-claude.md](06-webflow-mcp-claude.md).

## Comment choisir sa formation

Trois critères décisifs :

### Budget
- **0 €** → Webflow University ou Digidop (gratuites)
- **20 €** → Udemy (Rhalid Hidoun, 5h, accès à vie)
- **695 à 3 495 USD** → Flux Academy (Webflow Masterclass)
- **2 490 à 3 900 €** → Formations FR avec CPF (One Learn, Cube, Maestro)

### Format
- **Self-paced** (à son rythme) : Webflow Univ., Udemy, Flux Academy
- **Cohorte / live** : Le Wagon, Maestro, One Learn
- **Hybride** : One Learn (e-learning + classes virtuelles + coaching)

### Objectif
- **Découvrir** → Webflow University (Welcome to Webflow + Webflow 101)
- **Reconversion freelance** → Flux Academy ou One Learn
- **Reconversion CPF en France** → One Learn (RNCP37805) ou Maestro (RNCP41143)
- **Carrière agence/intégrateur** → Webflow Visual Developer Certification + Webflow Certified Partner
