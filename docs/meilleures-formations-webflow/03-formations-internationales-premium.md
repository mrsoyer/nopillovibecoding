## 3. Udemy — Option Budget

**Site** : [udemy.com/topic/webflow](https://www.udemy.com/topic/webflow/)

### Données clés
- **Prix** : 19,99 à 29,99 USD par cours (jusqu'à 199,99 USD pour les plus longs)
- **Cours FR phare** : Rhalid Hidoun — 5h on-demand, accès à vie, 19,99 €
- **Format** : asynchrone, accès à vie
- **Certification** : certificat de complétion Udemy (non reconnu employeur)

### Forces
- Très accessible budget
- Beaucoup de choix par instructeur, langue, niveau
- Idéal pour découvrir avant d'investir

### Faiblesses
- Qualité **inconsistante** d'un cours à l'autre
- Peut être obsolète (Webflow évolue vite)
- **Aucun mentorat**

## 4. Alternatives — YouTube + Mentors privés

### YouTube
- Des milliers de tutoriels gratuits problem-focused
- Vidéos typiquement 10 min, ciblées
- Excellent pour résoudre un point précis
- **Risque** : « tutorial hell », pas de structure
- Channels recommandés : Webflow officiel, Flux (Ran Segall), Pixel Geek, Timothy Ricks

### Mentors privés / Cohortes
- Sessions live avec feedback temps réel
- Approche personnalisée
- Vetting requis (qualité variable)
- **Très coûteux**, plus que les cours formels
- Pour designers avancés avec gros budget

## Comparaison Flux Academy vs Webflow University

D'après le blog officiel Flux Academy [« Webflow University vs The Webflow Masterclass »](https://www.flux-academy.com/blog/webflow-university-vs-the-webflow-masterclass-whats-the-difference) :

| Critère | Webflow University | Flux Masterclass |
|---------|--------------------|--------------------|
| **Prix** | Gratuit | 695-3 495 USD |
| **Approche** | Référentiel feature par feature | Pédagogie projet end-to-end |
| **Vitesse de progression** | Lente, pas structurée | Beaucoup plus rapide |
| **Mentorat** | Aucun | 8 sem. à 12 mois 1:1 |
| **Projets construits** | Tutoriels guidés | 3 vrais projets portfolio |
| **Carrière** | Pas de path | Path complet vers freelance |

**Conclusion** : Webflow University est **complémentaire**, pas concurrent. La plupart des étudiants Flux utilisent les deux.

## Pour qui ces formations EN sont mieux que les FR

Choisir l'international si :

- Vous voulez le **meilleur mentorat possible** (Flux > toute formation FR)
- L'**anglais** ne vous freine pas
- Vous **n'avez pas besoin du CPF** (entreprise paie, vous avez les fonds, ou vous êtes hors France)
- Vous voulez intégrer une **communauté internationale** (clients premium hors France)

Choisir une formation FR certifiante si :

- Vous voulez **mobiliser votre CPF**
- Vous voulez une **certification RNCP** reconnue France Travail
- Vous préférez l'accompagnement en français
- Vous ciblez le marché client français
