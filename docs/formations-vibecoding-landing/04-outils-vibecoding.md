## 1. Lovable

**Catégorie** : No-code AI builder

- **Cible** : Non-techs, designers, freelances
- **Pricing** :
  - Free : 5 crédits/jour
  - Pro : 25 $/mois (100 crédits/mois)
  - Business : 25 $/mois enhanced
- **Atouts** : MVP complet en 15 min, auth + DB intégrés, export GitHub, **50 % discount étudiant**
- **Faiblesses** : Tier gratuit limité, boucles de bugs si prompts imprécis
- **Verdict** : Démarrage idéal pour non-techs

## 2. Bolt.new

**Catégorie** : No-code AI builder rapide

- **Cible** : Prototypeurs, makers, éducateurs
- **Pricing** :
  - Free : 300 K tokens/jour, 1 M/mois
  - Pro : 25 $/mois (10 M tokens/mois)
  - Teams : 30 $/mois/user
- **Atouts** : "Le plus rapide du marché", rendu instantané, supporte React/Vue/Svelte/RN, free tier généreux
- **Faiblesses** : Tokens consommés vite, backend limité, coûteux pour projets complexes

## 3. Cursor ⭐ Référence dev

**Catégorie** : AI IDE (basé VS Code)

- **Cible** : Développeurs juniors et seniors
- **Pricing** :
  - Hobby (free, limité)
  - Pro : 20 $/mois
  - Pro+ : 60 $/mois
  - Ultra : 200 $/mois
- **Atouts** : Contrôle total sur le code, contexte projet entier, updates hebdo, choix de modèles AI
- **Faiblesses** : Courbe d'apprentissage non-techs, comportement imprévisible sur très gros projets, risque modifs non voulues

## 4. v0 by Vercel

**Catégorie** : UI generator (frontend only)

- **Cible** : Designers, frontend devs, équipes produit
- **Pricing** :
  - Free : 5 $ crédits/mois
  - Premium : 20 $/mois (Figma import)
- **Atouts** : Composants React/Tailwind impeccables, design pro, export clean
- **Faiblesses** : Frontend uniquement (pas de backend), 7 messages/jour en free
- **Combo idéal** : v0 (frontend) + Supabase (backend)

## 5. Replit

**Catégorie** : Cloud IDE multi-langages

- **Cible** : Débutants, étudiants, nomades digitaux
- **Pricing** :
  - Starter (free, limité)
  - Core : 20 $/mois (25 $ crédits)
  - Teams : 35 $/mois/user
- **Atouts** : Ultra-accessible, fonctionne sur mobile, 50+ langages, déploiement free, pair programming
- **Faiblesses** : Pricing "effort-based" (juin 2025) imprévisible, coûts pouvant escalader (350 $/jour reportés)

## 6. Windsurf

**Catégorie** : AI IDE (concurrent Cursor)

- **Cible** : Devs cherchant alternative Cursor
- **Pricing** :
  - Free : 25 crédits/mois
  - Pro : 15 $/mois (500 crédits)
  - Teams : 30 $/mois/user
- **Atouts** : 25 % moins cher que Cursor, agent Cascade intelligent, **autocomplétion illimitée en free**
- **Faiblesses** : Moins mature, free tier limité, changements de propriétaire récents

## 7. Claude Code ⭐ Recommandation principale 2026

**Catégorie** : AI agent autonome en terminal (Anthropic)

- **Cible** : Devs avancés, équipes voulant automatisation
- **Pricing** :
  - Pas de plan standalone
  - Inclus avec Claude Pro (~20 $/mois) ou Max (~90-200 $/mois)
- **Atouts** :
  - Travaille en autonomie pendant des heures
  - Raisonnement sophistiqué
  - Commits Git automatiques
  - Skills, subagents, MCP officiels
  - **Stack 2026 standard** pour SaaS (cf. tutoriels makerkit, aibuilderclub)
- **Faiblesses** : Pas de free tier, terminal-only, requiert maîtrise Git/CLI, quotas tokens partagés avec Claude.ai

> Voir [05-claude-code-landing-page.md](05-claude-code-landing-page.md) et [06-claude-code-supabase-saas.md](06-claude-code-supabase-saas.md) pour workflows.

## 8. Gemini CLI

**Catégorie** : AI agent terminal (Google)

- **Cible** : Devs CLI, étudiants, makers budget
- **Pricing** :
  - Free : **1 000 requêtes/jour**
  - AI Pro : 21,99 $/mois
  - Ultra : 274,99 $/mois
- **Atouts** : 1 M tokens context (le plus large), 1 000 req/jour gratuites, zero signup
- **Faiblesses** : Terminal-only, courbe d'apprentissage, moins intuitif

## 9. Base44

**Catégorie** : SaaS-builder spécialisé

- **Cible** : Entrepreneurs SaaS, freelances clients tools
- **Pricing** :
  - Free : 25 crédits/mois
  - Starter → Elite : 20 $ → 200 $/mois
- **Atouts** : Auth + BDD + paiements + analytics inclus, infrastructure SaaS pré-bâtie
- **Faiblesses** : Export code non disponible sur tous les plans, communauté limitée, use case étroit

## 10. Aider

**Catégorie** : Open source pair-programmer

- **Cible** : Devs privacy-conscious, makers budget, devs avancés
- **Pricing** : **Gratuit** (open source) — coûts API uniquement (0,01-0,10 $/feature avec cloud models)
- **Atouts** : 100 % gratuit, opération offline/locale, privacy totale, commits Git auto, modèles flexibles
- **Faiblesses** : Terminal-only, requiert compétences Git/CLI, config initiale, écosystème plus petit

## Recommandations par profil

### Profil "Je n'ai jamais codé"
1. **Démarrer** : Lovable ou Bolt.new (free)
2. **Si projet sérieux** : passer Pro Lovable (25 $/mois)
3. **Si MVP commercial** : Base44

### Profil "Je code mais pas en JS/TS"
1. **Démarrer** : Replit (browser, multi-langages)
2. **Évoluer** : Cursor une fois habitué à JS

### Profil "Je suis dev JS/TS confirmé"
1. **Daily driver** : Cursor ou Windsurf (Windsurf si budget serré)
2. **Tâches longues/automatisation** : Claude Code
3. **Composants UI rapides** : v0

### Profil "Je veux le maximum gratuit"
1. **Daily** : Gemini CLI (1 000 req/jour gratuites)
2. **Backup** : Bolt.new (free généreux)
3. **Open source** : Aider

### Profil "Je veux tout en privacy"
1. **Aider** + Ollama (modèles locaux)

## Anti-patterns à éviter

❌ **Démarrer par Cursor sans connaître JS/Git** — Trop de risque de tout casser sans s'en rendre compte
❌ **Tout faire en free tier** — Sur projets sérieux, le coût (20-30 $/mois) est dérisoire vs temps gagné
❌ **Multiplier les outils** — Choisir UN outil principal (Claude Code OU Cursor) et le maîtriser
❌ **Ignorer les MCP** — En 2026, MCP (Model Context Protocol) débloque énormément de productivité
