## Les 14 Éléments d'une Landing qui Convertit

### 1. Headline benefit-focused

**Quoi** : Headline principal communicant la value proposition en langage clair.

**Pourquoi** : Les visiteurs scannent rapidement, doivent savoir en 5 sec si l'offre les concerne.

**Comment** : Action-oriented, parlant aux besoins (pas aux features).

**Exemple** :
- ❌ "Enterprise Software Solutions"
- ✅ "Cut Project Management Time by 50%"

**Impact** : Bonne headline = +250-300 % conversion potentiel.

### 2. Subheadline qui renforce la promesse

**Quoi** : Headline secondaire clarifiant le **comment**.

**Implémentation** : Étendre la headline en parlant du mécanisme.

**Exemple** : "Automated workflows that let your team focus on strategy, not spreadsheets"

### 3. Above-the-fold épuré

**Quoi** : Zone immédiatement visible : headline + subheadline + visuel + CTA principal.

**Pourquoi** : La majorité des décisions de conversion se fait ici, **avant scroll**.

**Comment** : Balance text/visuels, white space, CTA proéminent.

### 4. Visuels et imagerie produit

**Quoi** : Images de qualité, produit en contexte ou usage réaliste.

**Pourquoi** : Le visuel se traite plus vite que le texte, réduit la charge cognitive.

**Anti-pattern** : Stock photos posées génériques.

### 5. Social proof / testimonials

**Quoi** : Témoignages, vidéos, logos clients, photos vraies.

**Impact** :
- Testimonials texte : **+34 %** conversion
- Reviews : **+270 %** conversion
- Video testimonials : **+80-86 %** vs texte
- **92 %** des consommateurs lisent les testimonials avant achat

**Exigences** : noms complets, photos pro, titres de poste, **résultats quantifiés** (ex: "Increased email revenue by 40% in 3 months").

### 6. Pain-point-focused copy

**Quoi** : Copy qui acknowledge les frustrations du visiteur AVANT de présenter la solution.

**Exemple** : "Tired of manual data entry? Our platform automates the entire process."

**Loss aversion > Gain framing** : "Stop losing customers to competitors" outperforme messages gain-focused.

### 7. Feature & Benefit Breakdown

**Pattern** : "Benefit: Save 10 hours weekly | Feature: AI-powered automation"

**Règle** : Lead with benefits, support with features.

### 8. Trust badges, certifications, garanties

**Quoi** : SSL badges, money-back, partner logos, industry certifications.

**Placement** : footer + près des CTAs.

**Critère** : Doivent être **current & légitimes** (sinon contre-productif).

### 9. CTA haute contraste

**Spécifications** :
- Couleurs contrastées
- Verbes d'action
- Répété stratégiquement (hero + mid-page + final)

**Exemples** :
- "Get Instant Access"
- "Start Free Trial"

**Case study cité** : Changer "Sign up for free" en "Trial for free" → **+104 %** trial start rate.

**Personnalisé > générique** : **+202 %** (Firework).

### 10. Forms minimaux

| Nombre de champs | Conversion |
|------------------|-----------|
| 3 champs | 10,1 % |
| 9 champs | 3,6 % |
| 5 champs ou moins | +120 % vs +5 |
| 11 → 4 champs | **+160 %** |

**Stats friction** :
- **81 %** abandonnent un form une fois commencé
- **67 %** ne reviennent jamais
- Causes : security concerns (29 %), longueur (27 %)

**Mobile autofill** : +24 % conversion vs forms sans autofill.

### 11. Mobile-optimised responsive

**Touch targets** : minimum 48×48 dp avec 8 dp d'espacement
**Mobile-first** : single column, tailles font lisibles, forms single-column

### 12. Page speed & technical foundation

| Délai chargement | Effet |
|-----------------|-------|
| 1 seconde | Référence |
| 5 secondes | -7 % conversion par sec, **conversion 3x inférieure** |
| > 3 secondes | **53 % mobile users abandonnent** |

**Optimisations** :
- Compresser images (WebP)
- Minimiser HTTP requests
- CDN
- Lazy loading (+22 % perceived performance)

**ROI calcul** : Pour $100K mensuels, passer de 5 → 2 sec = **+$21,000/mois** (Fleexy).

### 13. Liens trust internes/externes

**Quoi** : Liens vers about, case studies, certifications, FAQs — sans créer de distraction.

**Règle** : pas de navigation concurrente sur landing page.

### 14. AEO-friendly FAQs

**Quoi** : FAQs avec schema FAQPage structuré.

**Pourquoi** : Answer Engine Optimization → visibilité dans recherches IA et voice assistants.

**Stat** : 60 % des 25-34 ans utilisent voice search quotidiennement.

## Anti-patterns Mortels

### ❌ Forms à 11+ champs
Perte 60-80 % conversion. Demander email + nom max au premier contact, le reste plus tard.

### ❌ "Sign up for free" générique
Test : "Trial for free" → +104 %. "Get Instant Access" → +X. À tester.

### ❌ Stock photos génériques
Diminue trust. Préférer : screenshots produit, photos clients réels, vidéos use case.

### ❌ Multiples CTAs concurrents
Un seul CTA primaire, **répété** à plusieurs endroits. Pas trois CTAs différents.

### ❌ Headline tournée features
"Enterprise software solutions" perd contre "Cut PM time 50 %".

### ❌ Pas de social proof
Perte 19-34 % conversion immédiatement.

### ❌ Mobile non optimisé
Perte de 100 % de conversion potentielle mobile.

### ❌ Page > 3 secondes
53 % d'abandon mobile garanti.

### ❌ Reading level "université"
11,1 % conversion devient 5,3 %. Viser 5e-7e année.

### ❌ Generic testimonials
"thousands of satisfied customers" < "Increased email revenue by 40% in 3 months".

## Framework de Test

### Pré-requis A/B testing valide

- **1 000+ visiteurs hebdomadaires** sur la page (sinon pas de signal)
- **Durée minimum** : 2-4 semaines
- **MDE** (Minimum Detectable Effect) : 20-30 % pour petites entreprises (sinon variance trop forte)

### Méthodologie

1. **Tester un seul élément à la fois** (sequential, pas multivariate sauf si trafic massif)
2. **Hypothèse claire** avant le test
3. **Documentation** : noter résultats dans un journal de tests
4. Après gain > 86 % : ré-tester car peut être effet nouveauté

### Que tester en premier (effort/impact)

| Priorité | Élément | Impact attendu |
|----------|---------|---------------|
| **1** | Headline | +27 à +300 % |
| **2** | Form length | +120 % |
| **3** | CTA copy | +104 % |
| 4 | Hero image (réelle vs stock) | Variable |
| 5 | Social proof placement | +19-34 % |
| 6 | Page speed | +7 %/sec |
