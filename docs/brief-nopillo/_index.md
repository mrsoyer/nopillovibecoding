# Brief Nopillo — Questionnaire de decouverte

## Vue d'ensemble

Ce dossier contient les supports pour conduire l'entretien de **brief Nopillo** (P0 bloquante, voir `docs/formation-nopillo/02-enquetes-manquantes.md` section 1). Sans ce brief, impossible de cadrer le format (1j / 2j / 3j) ni d'adapter les modules a leur stack reelle.

## Pourquoi un brief structure

Nopillo n'apparait pas dans les recherches publiques. Toute la formation se decide sur ces reponses :

- Choix du format (duree, intra/inter, nombre de jours)
- Choix des modules a couvrir ou retirer (HubSpot ? Meta Ads ? Netlify ?)
- Choix de la difficulte (junior / mid / senior)
- Choix du tarif (CPF / OPCO / cash, intra vs inter)
- Choix du calendrier (J-7 setup necessaire)

Sans brief : risque eleve d'arriver J1 et decouvrir que la moitie du programme n'est pas pertinent.

## Comment l'utiliser

### Cas A — Entretien telephonique (recommande)

1. Caler 30-45 min de visio avec le decideur Nopillo (CEO ou head of design)
2. Avoir `01-questionnaire-decouverte.md` ouvert pour suivre
3. Remplir `02-fiche-synthese.md` en direct ou dans les 2h post-call
4. Le soir meme : appliquer la matrice `03-decision-format-formation.md`
5. J-7 : cocher la `04-checklist-pre-formation.md`

### Cas B — Questionnaire ecrit (si pas de creneau dispo)

1. Convertir `01-questionnaire-decouverte.md` en form Tally / Notion / Typeform
2. Envoyer avec deadline de reponse (J-21 minimum)
3. Faire un call de **clarification** de 15 min apres reception (toujours)
4. Remplir `02-fiche-synthese.md` puis enchainer comme cas A

### Cas C — Brief partiel (10 min, urgence)

Sauter au questionnaire et ne poser **que** les questions marquees `[CRITIQUE]`. Necessite un call de complement avant J-7 obligatoire.

```
J-14  Validation devis Nopillo
J-7   Checklist pre-formation cochee a 100%

```
