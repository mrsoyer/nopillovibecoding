## Formations gratuites (officielles Anthropic)

### 1. Anthropic Academy (lance 2026-03-02)

**URL** : https://anthropic.skilljar.com/
**Prix** : 100% gratuit, certificats inclus, sans compte Anthropic requis
**Format** : Self-paced, video + quiz + exercices pratiques
**Public** : Tous niveaux (parcours par cours)

**13 cours disponibles** :

| Cours | Niveau | Focus |
|---|---|---|
| Claude Code 101 | Debutant | Installation -> customisation avancee |
| Claude Code in Action | Intermediaire | Workflows pratiques |
| Introduction to Agent Skills | Intermediaire | Skills custom |
| Introduction to Claude Cowork | Tous | Equipe + collaboration |
| Building with the Claude API | Developpeur | Integration API + SDK |
| Claude with Google Vertex AI | Avance | Deployement cloud |
| (+ 7 cours autour MCP, prompting, agents) | Mixte | Ecosysteme complet |

**Avantages** :
- Source officielle = source de verite
- Mises a jour synchronisees avec releases produit
- Certificats reconnus
- Gratuit total, pas de freemium cache

**Limites** :
- Anglais uniquement
- Recent (mars 2026) -> retours communaute encore limites
- Pas de mentor / live

### 2. Coursera (gratuit)

**URLs** :
- "Claude Code: Software Engineering with Generative AI Agents" (Vanderbilt) : https://www.coursera.org/learn/claude-code
- "Claude Code in Action" : https://www.coursera.org/learn/claude-code-in-action
- "Building with the Claude API" : https://www.coursera.org/learn/building-with-the-claude-api

**Prix** : Gratuit (audit) ou ~49€/mois (certificat)
**Format** : Cours universitaire structure, evaluations notees
**Public** : Developpeurs ayant deja base programmation

**Particularite Vanderbilt** : "From skepticism to confident leverage" — angle pedagogique focalise sur changement de paradigme.

## Formations universitaires

### Vanderbilt University (Coursera)

**URL** : https://www.coursera.org/learn/claude-code
**Public** : Engineers experimentes
**Approche** : Universitaire, structuree en modules avec devoirs notes

## Formations payantes France (CPF eligibles)

### BGB Formation

**URL** : https://bgbformation.fr/formation-claude-code-cpf
**Prix** : 1490€ HT (~1788€ TTC)
**Duree** : 2 jours / 14h
**Format** : Distanciel et/ou presentiel
**Financement** : 4 voies (CPF + OPCO + France Travail + plan employeur), traitement CPF 11 jours ouvres
**Certification** : Qualiopi
**Public** : Developpeurs initie + curieux IA

### Human Coders

**URL** : https://www.humancoders.com/formations/claude-code
**Format** : Groupes 3-6 participants
**Villes** : Paris, Lyon, Bordeaux, etc. (multi-villes)
**Financement** : CPF + OPCO + Pole Emploi (Qualiopi certifie)
**Public** : Developpeurs avec base solide
**Plus** : Petits groupes = forte interaction
**Moins** : Tarif non public (devis sur demande)

### NoCode IA Toulouse

**URL** : https://nocodetoulouse.fr/formation-claude-code-vibe-coding/
**Duree** : 5 jours
**Approche** : Vibe coding focus, A->Z creation app autonome
**Public** : Entrepreneurs / freelances / non-devs
**Plus** : Format long = approfondissement, focus vibecoding moderne
**Moins** : Geographique (Toulouse)

### formation-claudecode.com

**URL** : https://www.formation-claudecode.com/
**Geographies** : France, Suisse, Belgique
**Tagline** : "Developpez 10x plus vite avec l'IA"
**Public** : Developpeurs production
**Note** : Specialiste pure-player Claude Code
