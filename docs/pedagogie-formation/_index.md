# Pedagogie Workshop Tech 1-3 Jours

> Reference pedagogique pour concevoir et animer des formations tech (Webflow, Claude Code, Ads, etc.) sur des formats de 1 a 3 jours.
> Base sur l'andragogie de Knowles, le modele 70-20-10, et les best practices des workshops agiles 2026.

## Public cible

- Formateurs Nopillo / freelances animant des workshops tech
- Personnes designant un programme de formation pour adultes en environnement professionnel
- Chefs de projet validant une offre de formation cliente

## Trois principes clefs

1. **70% pratique / 20% social / 10% theorie** — l'inverse du reflex slide-heavy
2. **Self-directed + experience-based** — adultes apprennent en faisant, pas en ecoutant
3. **Skill check before/after** — mesurer l'effet du workshop, pas la satisfaction

## Comment utiliser cette doc

- Tu designes un workshop ? Lis `01-overview.md` puis le format cible (`02`, `03` ou `04`).
- Tu animes ? Lis `05-techniques-animation.md` + `07-materiel-pedagogique.md`.
- Tu mesures l'impact ? Lis `06-evaluation.md`.
