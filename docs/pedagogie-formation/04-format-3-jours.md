# 04 — Format 3 jours : Alternance + Projet final

> Format **maitrise complete** : 3 jours, alternance fine theorie/pratique/projet, ratio 15% theorie / 85% pratique. Pour transformer un junior en operationnel autonome ou refondre la pratique d'une equipe complete.

## Cas d'usage

- Transformation d'une equipe : passer de "manuel" a "Claude Code + MCP" sur tous les workflows
- Onboarding complet d'un junior agence : webflow + Claude + ads + tracking
- Certification interne ("operateur Nopillo niveau 1") avec evaluation reelle

## Contraintes & opportunites du 3 jours

| Contrainte | Opportunite |
|------------|-------------|
| Investissement lourd (cher, hors prod) | Vraie autonomie a la sortie, mesurable |
| Risque de saturation J3 | Possibilite vrai projet final note |
| Necessite formateur(s) experimente(s) | Permet co-animation + 1-to-1 cible |

## Structure type (24h, dont 18h actives)

### Jour 1 : Fondations + Reproduction

```
09:00-09:30  Accueil + skill check pre (30min)
09:30-10:30  Vue d'ensemble methodologie + framework (1h)
10:30-10:45  PAUSE
10:45-12:30  Demo + reproduce piece 1 (1h45)
12:30-14:00  DEJEUNER
14:00-15:30  Pratique encadree variation 1 (1h30)
15:30-15:45  PAUSE
15:45-17:00  Code review + retour formateur (1h15)
17:00-17:30  Retro J1 + brief J2 (30min)
```

### Jour 2 : Approfondissement + Cas perso

```
09:00-09:30  Recap interactif J1 (30min)
09:30-10:30  Theorie ciblee 2 (1h)
10:30-10:45  PAUSE
10:45-12:30  Pratique cas perso piece 2 (1h45)
12:30-14:00  DEJEUNER
14:00-15:30  Demo croisee participants + feedback (1h30)
15:30-15:45  PAUSE
15:45-17:00  Brief projet final + setup (1h15)
17:00-17:30  Retro J2 + planning J3 (30min)
```

### Jour 3 : Projet final + Certification

```
09:00-09:30  Stand-up + objectifs J3 (30min)
09:30-12:30  Realisation projet final (3h, formateur en support 1-to-1)
12:30-14:00  DEJEUNER
14:00-16:00  Finition + tests + QC (2h)
16:00-16:30  PAUSE LONGUE
16:30-17:30  Demo finale + jury / feedback (1h)
17:30-18:00  Skill check post + bilan + planning suivi (30min)
```

**Total actif** : 18h sur 24h (75%).
**Theorie pure** : 2h cumulees (8%).
**Pratique** : 13h+ (54%).
**Social/projet** : 5h+ (21%).

## Le projet final (cle de voute du 3 jours)

Le 3 jours **se justifie par le projet final**. Sans projet final, c'est un 2 jours etire.

### Caracteristiques projet final

- **Realisable en 5-7h** par un participant equipe + accompagne
- **Reel** : sur le brief client perso ou cas Nopillo donne
- **Evaluable** : checklist de criteres objectifs (perf, accessibilite, tracking, design)
- **Demonstrable** : presentation 5 min en fin de J3
- **Reutilisable** : devient un template / portefolio personnel

### Exemples projet final

| Workshop 3j | Projet final |
|-------------|--------------|
| Nopillo full-stack | Landing complete : design + integration + form HubSpot + tracking GA4 + Meta CAPI |
| Claude Code agency | Skill agency reutilisable + 3 skills perso + CDC d'un projet client |
| Ads landing avance | 3 variantes A/B testees, 1 par canal (Google/Meta/LinkedIn) avec setup conversion |

## Alternance fine theorie/pratique/projet

Sur 3 jours, le ratio cible :

```
Jour 1 : 30% theorie / 70% reproduce
Jour 2 : 15% theorie /  85% adapter (cas perso)
Jour 3 :  0% theorie / 100% projet final + demo
```

L'evolution = **autonomisation progressive** : main tenue J1, lachee J3.

## Demarrage J2 et J3 : stand-up

Le J2 et J3 ouvrent par un **stand-up agile** de 15 min :
- Chacun dit : "hier j'ai fait X, aujourd'hui je vais faire Y, j'ai besoin de Z"
- Le formateur identifie les blocages et adapte la journee

Permet de detecter tot ceux qui decrochent.

## Pre-work 3 jours

Plus exigeant :
- [ ] Tous setup du 1j et 2j
- [ ] Lecture preparatoire : 2-3 articles ou 1 doc complete (1-2h)
- [ ] Brief projet final perso valide par le formateur (J-7)
- [ ] Acces compte client / sandbox / staging valide

Mail J-14 + relance J-7 + relance J-3 + relance J-1.

## Animation : co-animation recommandee

Sur 3 jours, **2 formateurs** est l'ideal :
- 1 formateur lead (anime, fait les demos)
- 1 formateur support (1-to-1 ceux qui decrochent, aide projet final J3)

Sinon, sur J3 surtout, le formateur seul ne peut pas suivre 5+ projets en parallele.

## Skill check 3 jours (Kirkpatrick complet)

**Niveau 1 (reaction)** — fin J3 (5 min) :
- NPS + 2 questions ouvertes (best/worst)

**Niveau 2 (apprentissage)** — pre J1 + post J3 (10 min) :
- Test technique 10 questions
- Auto-eval 1-5 par competence
- Mesure delta

**Niveau 3 (comportement)** — J+30, J+60 (mail) :
- "Avez-vous applique sur un projet reel ? Lequel ? Resultat ?"
- Cible : > 70% des participants ont applique

**Niveau 4 (resultat)** — J+90 (mail ou call) :
- "Impact mesurable sur votre activite ?" (temps gagne, projets livres)
- Cible : ROI explicite (ex: 10h gagnees sur prochain projet)

## Materiel pedagogique 3 jours

| Support | Format | Utilite |
|---------|--------|---------|
| Slide deck | 50-80 slides max | 2-3 blocs theorie cibles |
| Repo template multi-niveaux | GitHub avec branches | Niveaux reproduce/adapter/projet |
| Bibliotheque prompts | Markdown indexe | Toutes les variantes de cas |
| Cheatsheet | PDF 2-4 pages | Reference quotidienne post-formation |
| Grille evaluation projet | Markdown | Critere objectifs J3 |
| Office hours | 4h offert sur 1 mois | Suivi mise en pratique reelle |
| Communaute | Slack / Discord acces | Pair learning continu |

## Tarification indicative

Format 3 jours Nopillo type :
- 1 participant : 3 800 € HT
- 2-4 participants : 2 500 €/pers
- Equipe (5-8) : 14 000 € forfait
- Office hours suivi : 4h inclus
- Acces communaute : 6 mois inclus

## Pieges specifiques 3 jours

| Piege | Fix |
|-------|-----|
| Saturation J3 | Projet final motivant + breaks longues |
| 1 seul formateur sur J3 | Co-animation indispensable au-dela de 4 participants |
| Projet final hors-sol | Brief valide par formateur en J-7 |
| Pas de Kirkpatrick 3-4 | Mail J+30 et J+90 systematiques |
| Pas de communaute apres | Slack/Discord = ancrage 70-20-10 a long terme |
