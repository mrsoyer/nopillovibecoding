# 03 — Format 2 jours : Theorie + Pratique

> Format **approfondissement** : 2 jours, alternance theorie consolidee + pratique guidee, ratio 20% theorie / 80% pratique. Pour quelqu'un qui veut **maitriser** un workflow et le re-jouer en autonomie.

## Cas d'usage

- Mise en autonomie d'un freelance sur la stack Nopillo
- Formation d'un junior pour qu'il livre seul des landings standards
- Approfondissement d'une thematique (ads, tracking, integration)

## Contraintes & opportunites du 2 jours

| Contrainte | Opportunite |
|------------|-------------|
| 2 jours hors prod -> cher | Permet 2 livrables, donc validation reelle |
| Energie a gerer sur 2 jours | Possible de "dormir" sur les concepts |
| Risque "decroche" J2 matin | Demarrer J2 par recap interactif |

## Structure type (16h, dont 12h actives)

### Jour 1 : Decouvrir + Reproduire

```
09:00-09:30  Accueil + skill check (30min)
09:30-10:30  Theorie : framework / methodologie (1h)
10:30-10:45  PAUSE
10:45-12:30  Demo + reproduce (Bloc 1) — pair programming (1h45)
12:30-14:00  DEJEUNER
14:00-15:30  Pratique encadree : variation 1 (1h30)
15:30-15:45  PAUSE
15:45-17:00  Code review groupe + Q/R (1h15)
17:00-17:30  Retro J1 + brief J2 (30min)
```

### Jour 2 : Adapter + Livrer

```
09:00-09:30  Recap interactif J1 (quiz pair) (30min)
09:30-10:30  Bloc theorie 2 : cas avances (1h)
10:30-10:45  PAUSE
10:45-12:30  Pratique : cas perso participant (1h45)
12:30-14:00  DEJEUNER
14:00-16:00  Realisation livrable final (2h)
16:00-16:15  PAUSE
16:15-17:00  Demo croisee + feedback formateur (45min)
17:00-17:30  Skill check post + planning suivi (30min)
```

**Total actif** : 12h sur 16h (75%).
**Theorie pure** : 2h max (12.5%).
**Pratique** : 9h+ (56%).
**Social** (demo, code review, pair) : 3h+ (18%).

## Les 2 livrables

Le 2 jours = **1 livrable simple J1 + 1 livrable adapte au cas perso J2**.

### Exemples

| Workshop 2j | Livrable J1 | Livrable J2 |
|-------------|-------------|-------------|
| Claude Code intermediaire | 1 skill personnel deploye | 1 skill agency commit dans repo equipe |
| Webflow + HubSpot | Page Webflow + form HubSpot mock | Landing perso + form HubSpot prod + tracking |
| Google Ads landing | Landing Quality Score 8+ template | Landing perso QS 8+ + setup conversion CAPI |

## Demarrage J2 : recap interactif

Le piege J2 : 30% du J1 est oublie au reveil. Le recap doit etre **actif**, pas un slide review.

Format recap (30 min) :
1. **Quiz pair** (10 min) : binomes se posent 5 questions sur J1
2. **Speed-share** (10 min) : chacun resume J1 en 1 min a son binome
3. **Q/R formateur** (10 min) : ce qui a coince est verbalise

## Alternance theorie/pratique

Bloc type recommande (90 min) :
- 15 min concept + demo (theorie active)
- 60 min pratique en pair (experience)
- 15 min code review groupe (social)

Repete x6-8 sur les 2 jours.

## Pre-work 2 jours

Plus ambitieux que le 1 jour :
- [ ] Tous les setup du 1 jour
- [ ] Lecture obligatoire : 1 article ou 1 chapitre doc (30 min)
- [ ] Brief perso detaille (le projet reel sur lequel le J2 sera applique)
- [ ] Cas concurrent a documenter (en option, "scout")

Envoyer J-7 + relance J-3 + relance J-1.

## Animation J2 : maintenir l'energie

Le J2 a un risque "soufflet" (relachement matin). Contre-mesures :

| Moment | Risque | Contre-mesure |
|--------|--------|--------------|
| J2 9h | "On a deja vu ca" | Recap interactif, pas slide |
| J2 11h | Saturation theorie | Cap 1h theorie max |
| J2 14h | Endormissement | Energizer 5 min ("paper plane game" ou variant) |
| J2 16h | Pression livrable | Aide formateur 1-to-1 prioritaire |

## Skill check 2 jours

**Avant J1** (5 min) :
- 5 questions techniques (skills cibles)
- Auto-eval 1-5 par competence

**Mi-parcours fin J1** (5 min, optionnel) :
- "Note de 1 a 5 ta confiance pour faire X seul demain"
- Identifier ceux qui decrochent -> aide ciblee J2

**Apres J2** (15 min) :
- Re-poser les 5 questions techniques
- Re-eval 1-5 par competence
- Question ouverte : "Que feras-tu differemment chez toi ?"

Le **delta moyen** sur l'auto-eval = KPI principal. Cible : +2 points sur 5.

## Materiel pedagogique 2 jours

| Support | Format | Utilite |
|---------|--------|---------|
| Slide deck | 40-60 slides | 2 blocs theorie |
| Repo template + variantes | GitHub | Reproduce + adapter |
| Prompts pre-ecrits | Markdown indexe | Tous les cas standards |
| Checklist QC livrable | Markdown | Auto-validation J2 |
| Cheatsheet | PDF 1-2 pages | Reference apres workshop |
| Office hours | 2h offert post | Suivi reel |

## Tarification indicative

Format 2 jours Nopillo type :
- 1 participant : 2 200 € HT
- 2-4 participants : 1 500 €/pers
- Equipe (5-8) : 8 500 € forfait
- Office hours suivi : 2h inclus

## Pieges specifiques 2 jours

| Piege | Fix |
|-------|-----|
| Refaire J1 le J2 | Recap actif < 30 min, on enchaine |
| 2 livrables identiques | Le J2 doit etre **personnel** au participant |
| Rythme uniforme | Alterner theorie/pratique/demo toutes les 90 min |
| Pas de "cas perso" | Exiger le brief perso au pre-work |
